Class: BootstrapGridRendererGroup

p-waves/dist/layouts/list.BootstrapGridRendererGroup()

An example "Renderer group" using Boostrap grids. A renderer group is a "meta-renderer" that relays events to several renderers. This is useful for breaking-up components in a complex renderer.
