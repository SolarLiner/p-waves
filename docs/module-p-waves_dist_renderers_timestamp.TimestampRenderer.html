Class: TimestampRenderer

p-waves/dist/renderers/timestamp.TimestampRenderer()

Example implementation of elapsed time over total time indicator. For now, you can skip rendering by providing an empty function, but function calls are wasted. In future versions, control flags can be added to prevent rendering.
